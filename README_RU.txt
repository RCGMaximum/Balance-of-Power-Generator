Версия 1.0 | Май 2026
============================================

НАЗНАЧЕНИЕ
----------
Программа создаёт файлы для моддинга механики Balance of Power 
в Hearts of Iron IV версия 1.18.2 (Возможно работает и на более ранних версиях, не проверял)

ЧТО ГЕНЕРИРУЕТ
--------------
- Основной файл BoP (common/bop/)
- Локализацию на русском и/или английском (localisation/)
- код GFX для иконок сторон (interface/)
- Категории решений (common/decisions/categories/)
- Решения (common/decisions/)
- Подробный README с инструкцией по установке

КАК ИСПОЛЬЗОВАТЬ
----------------
1. Запустите BOP_Generator.exe
2. Выберите язык интерфейса (1 - Русский, 2 - English)
3. Введите тег вашей страны (например, GER) или нажмите Enter
4. Следуйте инструкциям на экране
5. После завершения в папке "generated_bop" появятся все файлы
6. Скопируйте их в папку вашего мода
7. В common/national_focus/ВАШ_ТЕГ.txt добавьте код активации
   (он будет показан в конце работы программы и сохранён в сгенерированный README)

ПРАВИЛА ЗАПОЛНЕНИЯ
------------------
- Все значения от -1.0 до 1.0 (игра не примет другие)
- min всегда меньше max (например: -0.8 < -0.5)
- Диапазоны идут без разрывов (max одного = min следующего)
- Диапазоны не пересекаются
- Левая сторона: от -1.0 до нейтрального
- Правая сторона: от нейтрального до +1.0
- Нейтральный диапазон строго между сторонами
- Все ID должны быть уникальными
- Дробные числа: через точку или запятую (0.8 или 0,8)

ДЛЯ ВВОДА КОДА (on_activate, on_deactivate, решения)
---------------------------------------------------
- Нажмите Enter на пустой строке — поле останется пустым
- Напишите DONE (в любом регистре) и нажмите Enter — завершить ввод
- Вводите код построчно, затем DONE для завершения

СОВМЕСТИМОСТЬ
