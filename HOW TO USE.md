\============================================

ИНСТРУКЦИЯ ПО УСТАНОВКЕ / INSTALLATION GUIDE

\============================================

РУССКИЙ

\-------

1. Поместите BOP\_Generator.exe в любую удобную папку
1. Запустите программу двойным кликом
1. Следуйте инструкциям на экране
1. Сгенерированные файлы появятся в папке "generated\_bop"
1. Скопируйте содержимое этой папки в папку вашего мода

Для работы программы НЕ требуется:

- Установка Python
- Установка дополнительных библиотек
- Права администратора
- Подключение к интернету

ENGLISH

\-------

1. Place BOP\_Generator.exe in any convenient folder
1. Run the program by double-clicking
1. Follow the on-screen instructions
1. Generated files will appear in the "generated\_bop" folder
1. Copy the contents of this folder into your mod folder

The program does NOT require:

- Python installation
- Additional libraries
- Administrator rights
- Internet connection
